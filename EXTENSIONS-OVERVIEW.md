## üöÄ Installation Guide

### For Chrome

**Job Application Tracker:**
1. Extract `job-tracker-extension.zip`
2. Open `chrome://extensions/`
3. Enable "Developer mode" (top-right toggle)
4. Click "Load unpacked"
5. Select the `job-tracker-extension` folder

**LinkedIn Connection Finder:**
1. Extract `linkedin-connections-extension.zip`
2. Open `chrome://extensions/`
3. Enable "Developer mode" (top-right toggle)
4. Click "Load unpacked"
5. Select the `linkedin-connections-extension` folder
6. **Important**: Make sure you're logged into LinkedIn!

### For Firefox

**Job Application Tracker:**
1. Extract `job-tracker-extension.zip`
2. Open `about:debugging#/runtime/this-firefox`
3. Click "Load Temporary Add-on..."
4. Select `manifest.json` in the folder

**LinkedIn Connection Finder:**
1. Extract `linkedin-connections-extension.zip`
2. Open `about:debugging#/runtime/this-firefox`
3. Click "Load Temporary Add-on..."
4. Select `manifest.json` in the folder
6. **Important**: Make sure you're logged into LinkedIn!

**Firefox Note**: Temporary extensions are removed when Firefox closes. You'll need to reload them each time.

## üõ†Ô∏è Troubleshooting

### Job Application Tracker

**Problem**: Floating button not appearing
- Wait 2-3 seconds for page to load
- Refresh the page
- Check you're on a supported job site
- Verify extension is enabled

**Problem**: Can't update a job
- Click the extension icon to use dashboard
- Try clearing cache and reloading
- Check browser console for errors (F12)

**Problem**: Follow-up notifications not working
- Check browser notification permissions
- Verify extension has notification permission
- Make sure date is in the future

### LinkedIn Connection Finder

**Problem**: Banner not showing
- **Make sure you're logged into LinkedIn!**
- Check extension settings (banner may be disabled)
- Wait 2-3 seconds for detection
- Refresh the page

**Problem**: No connections found
- This is normal if you don't have connections there
- Click "Search Manually" for 2nd/3rd degree
- Expand your LinkedIn network over time

**Problem**: Not detecting company
- Try refreshing the page
- Check you're on a supported job site
- Some company names may not match exactly
- Use manual search as backup
